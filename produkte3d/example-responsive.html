Responsive example

STOP

In this example we are programmatically changing width of #view360 div.
CSS we are using: #view360{
width:100%;
min-height:200px;
} View360 take width of #view360 and automatically adjust height.
document.addEventListener( "DOMContentLoaded", onLoaded );

function onLoaded (){

var view = new View360();
view.setMode("responsive");
view.setImagesDirectory( "./products/sami-fullsize" );
view.start( document.getElementById("view360") );

}
