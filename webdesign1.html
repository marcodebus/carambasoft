Webseite für CS Kälte Klima

Kunde: CS Klimatechnik

Ersteller: Marco Debus

Datum: 30. September, 2019

Dauer Erstellung: 14 Tage

Besonderheiten: Animation, Datenbankverknüpfung, Backend

Haida | CS Kälte Klima

Wir haben für die Firma Haida, CS Kälte Klimatechnik eine neue Webseite erstellt. Diese wird mit dem PHP Framework Codeigniter betrieben. Sämtliche Verwaltungsvorgänge werden über das Backend erledigt.
Die Kontaktformulare werden über eine API Schnittstelle zu Codeigniter Betrieben, hierbei werden sämtliche Eingabedaten auf Vieren und Schadenssoftware geprüft.
