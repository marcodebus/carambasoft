Webseite für Help Animals

Kunde: Darleene Bergmann

Ersteller: Marco Debus

Datum: 22. Dezember, 2019

Dauer Erstellung: 4 Wochen

Besonderheiten: Animation, Datenbankverknüpfung, Backend | Social Media

Sorglos.Immo | Sorglosenergie

Wir haben für die Firma Sorglos Immo GbR, eine neue Webseite erstellt. Diese wird mit dem PHP Framework Codeigniter betrieben. Sämtliche Verwaltungsvorgänge werden über das Backend erledigt.
Die Kontaktformulare werden über eine API Engine zu Codeigniter verknüpft, hierbei werden sämtliche Eingabedaten auf Vieren und Schadenssoftware geprüft und Verschlüsselt.

Die Seite ist Skalierbar und kann läuft aktuell über 4 V-Server die im Cluster verbunden sind um beim hohen Traffic keine Fehler zu verursachen.
